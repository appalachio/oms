# Organization Management System (oms)

OMS is a full stack, Ruby on Rails web application designed to help nonprofits and small organizations meet their online and technology needs. Currently, this includes a content management system (CMS) and user management.

## Contributing

Contributors are welcomed and encouraged! Please first review the [Code of Conduct](https://github.com/appalachio/oms/blob/main/code_of_conduct.md).

The tests and checks that must pass for any changes to be accepted can be run by calling:

```console
bin/test
```

from the project's root directory.
